# Shadow Pirate

## How to play
* Press SPACE to start
* Use arrow keys to move
* Press S to attack

## Rules of the game
Jump to dodge the obstacles as they come.
Level 1 = Reach the ladder!
Level 2 = Reach the treasure!
Ensure the player health does not reach 0%.

Happy playing!
